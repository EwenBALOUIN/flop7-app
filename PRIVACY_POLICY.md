# Politique de Confidentialité - Flip 7

**Dernière mise à jour :** 1er janvier 2025

## Introduction

Flip 7 ("nous", "notre", "l'application") est une application mobile développée pour gérer les parties du jeu de société Flip 7. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos informations lorsque vous utilisez notre application.

## Collecte de Données

### Données Stockées Localement

Flip 7 est une application **entièrement hors ligne** qui stocke toutes les données uniquement sur votre appareil. Nous collectons et stockons les informations suivantes localement via AsyncStorage :

- **Noms des joueurs** : Les noms que vous entrez pour identifier les joueurs dans une partie
- **Scores des parties** : Les scores de chaque tour et le total de chaque joueur
- **Métadonnées des parties** : Nom de la partie (optionnel), date de création et de dernière modification

### Aucune Collecte de Données Personnelles

Nous **ne collectons PAS** :
- Informations d'identification personnelle (adresse e-mail, nom réel, numéro de téléphone)
- Données de localisation
- Informations de paiement
- Données biométriques
- Identifiants publicitaires
- Historique de navigation ou de recherche
- Contacts ou fichiers de votre appareil

### Aucune Transmission de Données

Flip 7 **ne transmet aucune donnée** :
- Aucune connexion réseau n'est établie
- Aucune donnée n'est envoyée à des serveurs externes
- Aucune analyse ou suivi n'est effectué
- Aucune publicité n'est affichée

## Utilisation des Données

Les données stockées localement sont utilisées uniquement pour :
- Permettre la gestion des parties de Flip 7
- Afficher les scores et les classements
- Conserver l'historique des parties
- Fonctionner en mode hors ligne

## Partage de Données

**Nous ne partageons aucune donnée** avec des tiers. Toutes les informations restent sur votre appareil et ne sont jamais transmises à l'extérieur.

## Sécurité des Données

Les données sont stockées localement sur votre appareil via AsyncStorage, qui est une solution de stockage sécurisée fournie par React Native. Aucune donnée n'est chiffrée car elle ne contient que des informations de jeu non sensibles (noms de joueurs et scores).

## Suppression des Données

Vous pouvez supprimer toutes les données de l'application à tout moment en :
- Supprimant une partie individuelle depuis l'écran d'accueil
- Désinstallant l'application (toutes les données seront supprimées)

## Droits des Utilisateurs

Conformément aux réglementations sur la protection des données (RGPD, CCPA, etc.), vous avez le droit de :
- **Accéder** à vos données : Toutes vos données sont visibles dans l'application
- **Supprimer** vos données : Vous pouvez supprimer les parties à tout moment
- **Portabilité** : Vous pouvez exporter manuellement vos données en consultant les parties dans l'application

## Modifications de cette Politique

Nous nous réservons le droit de modifier cette politique de confidentialité. Toute modification sera indiquée par une mise à jour de la date "Dernière mise à jour" en haut de ce document.

## Conformité

Cette application est conforme aux exigences des stores d'applications :
- **Google Play** : Conforme aux exigences de confidentialité des données
- **Apple App Store** : Conforme aux directives de confidentialité d'Apple

## Contact

Pour toute question concernant cette politique de confidentialité ou le traitement de vos données, vous pouvez :
- Ouvrir une issue sur le dépôt GitHub du projet
- Consulter le code source de l'application (open source)

## Résumé

**En bref :** Flip 7 est une application 100% hors ligne qui stocke uniquement les noms des joueurs et les scores localement sur votre appareil. Aucune donnée n'est collectée, transmise ou partagée. Votre vie privée est respectée.

---

*Cette politique de confidentialité est valable pour la version 1.0.0 et les versions ultérieures de l'application Flip 7.*
