# Guide de Publication sur les Stores

Ce document contient les informations nécessaires pour publier Flip 7 sur Google Play et Apple App Store.

## Politique de Confidentialité

### URL GitHub Pages
Une fois GitHub Pages activé, l'URL de votre politique de confidentialité sera :
```
https://[votre-username].github.io/flop7-app/privacy-policy.html
```

### URL Alternative (Markdown)
Si vous préférez utiliser le fichier Markdown directement :
```
https://github.com/[votre-username]/flop7-app/blob/main/PRIVACY_POLICY.md
```

## Google Play Store

### Informations Requises

1. **Politique de Confidentialité** (OBLIGATOIRE)
   - URL : `https://[votre-username].github.io/flop7-app/privacy-policy.html`
   - Emplacement : Google Play Console > Politique > Politique de confidentialité

2. **Données Collectées**
   - ✅ Aucune donnée personnelle collectée
   - ✅ Aucune donnée partagée avec des tiers
   - ✅ Aucune connexion réseau
   - ✅ Stockage local uniquement (AsyncStorage)

3. **Permissions**
   - Aucune permission spéciale requise
   - L'application fonctionne sans accès réseau, contacts, localisation, etc.

### Déclaration de Confidentialité Google Play

Répondez aux questions suivantes dans Google Play Console :

- **Collecte de données** : Non
- **Partage de données** : Non
- **Sécurité des données** : Les données sont stockées localement uniquement
- **Suppression des données** : Les utilisateurs peuvent supprimer toutes les données en désinstallant l'application

## Apple App Store

### Informations Requises

1. **Privacy Policy URL** (OBLIGATOIRE)
   - URL : `https://[votre-username].github.io/flop7-app/privacy-policy.html`
   - Emplacement : App Store Connect > App Information > Privacy Policy URL

2. **Privacy Practices**
   - ✅ Aucune collecte de données
   - ✅ Aucun suivi
   - ✅ Aucune publicité

### Déclaration de Confidentialité Apple

Dans App Store Connect, déclarez :
- **Données collectées** : Aucune
- **Données liées à l'utilisateur** : Aucune
- **Données utilisées pour le suivi** : Aucune
- **Données liées à l'identité** : Aucune

## Checklist de Publication

### Avant la soumission

- [ ] GitHub Pages activé et fonctionnel
- [ ] Politique de confidentialité accessible via l'URL
- [ ] Tous les textes de l'application vérifiés
- [ ] Icônes et captures d'écran préparés
- [ ] Description de l'application rédigée
- [ ] Mots-clés définis

### Google Play

- [ ] Compte développeur créé (frais unique de $25)
- [ ] Application signée avec une clé de release
- [ ] Politique de confidentialité ajoutée
- [ ] Déclaration de confidentialité complétée
- [ ] Catégorie sélectionnée : Jeux > Jeux de société
- [ ] Classification de contenu complétée

### Apple App Store

- [ ] Compte développeur Apple créé ($99/an)
- [ ] Certificats et profils de provisioning configurés
- [ ] Privacy Policy URL ajoutée
- [ ] Privacy practices déclarées
- [ ] Catégorie sélectionnée : Games > Board
- [ ] Classification d'âge : 4+ (pas de contenu inapproprié)

## Notes Importantes

1. **Transparence** : La politique de confidentialité doit être accessible avant le téléchargement
2. **Mise à jour** : Si vous modifiez la politique, mettez à jour la date dans le document
3. **Conformité** : Assurez-vous que votre application respecte réellement ce qui est déclaré dans la politique

## Support

Pour toute question sur la publication :
- Consultez la documentation officielle de Google Play Console
- Consultez la documentation officielle d'App Store Connect
- Ouvrez une issue sur le dépôt GitHub
