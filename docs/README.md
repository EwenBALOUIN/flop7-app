# Documentation GitHub Pages

Ce dossier contient les fichiers HTML pour GitHub Pages.

## Fichiers

- `index.html` - Page d'accueil de la documentation
- `privacy-policy.html` - Politique de confidentialité complète

## Activation de GitHub Pages

Pour activer GitHub Pages sur votre dépôt :

1. Allez dans **Settings** > **Pages** de votre dépôt GitHub
2. Sous **Source**, sélectionnez **GitHub Actions**
3. Le workflow `.github/workflows/pages.yml` déploiera automatiquement les fichiers du dossier `docs/`

## URL de la politique de confidentialité

Une fois GitHub Pages activé, votre politique de confidentialité sera accessible à :
- `https://[votre-username].github.io/flop7-app/privacy-policy.html`

## Utilisation pour les stores

Vous pouvez utiliser cette URL dans :
- **Google Play Console** : Section "Politique de confidentialité"
- **Apple App Store Connect** : Section "Privacy Policy URL"
